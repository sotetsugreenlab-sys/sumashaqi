# スマシャキ

栽培作業の当日スケジュールを確認するためのシングルページWebアプリ。

## 機能

### 当日の作業カード

今日の日付を基準に、播種から何日目かを逆算し、以下の4タスクをカード表示する。

| タスク | 播種からの日数 |
|--------|--------------|
| 30穴へ移植 | 5日目 |
| 6穴へ移植 | 14日目 |
| 3穴へ移植 | 22日目 |
| 刈取 | 32日目 |

各カードには播種日と、それまでの移植履歴（日付）も表示される。

### 栽培日程計算機

- **播種日 → 各日程**：播種日を入力すると、30穴・6穴・3穴・刈取の各日付を計算
- **刈取日 → 播種日**：刈取日を入力すると、播種日および各日程を逆算（刈取は播種から32日固定）

### イレギュラーイベントバナー

Google スプレッドシートからCSVを取得し、当日に登録されたイベントをページ上部にバナー表示する。

- シート管理方法：A列に日付（例：`2025-04-16`）、B列に内容を入力

## ファイル構成

```
sumashaqi_daily_schedule_v3.html   # アプリ本体（HTML/CSS/JS 一体型）
```

## 技術仕様

- 純粋なHTML/CSS/JavaScript（外部フレームワークなし）
- Google Sheets の公開CSV URL へ `fetch` してイベントを取得
- `apple-mobile-web-app-capable` 対応（ホーム画面追加でPWA的に使用可能）
